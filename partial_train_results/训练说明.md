# 训练执行说明

## 当前状态

已启动 lambda=0.1 的 CROWN-IBP 训练（后台运行中）：
```bash
python train.py --lambda-weight 0.1 --bound-method CROWN-IBP --epochs 5
```

## 重要说明

### ⚠️ 训练时间预估

**CROWN-IBP 方法在 CPU 上非常慢**：
- 每个 epoch 约需 30-60 分钟
- 5个 epochs 总计可能需要 **2.5 - 5 小时**
- 三次训练（lambda=0.1, 0.5, 1.0）总计可能需要 **7.5 - 15 小时**

### 💡 建议的替代方案

如果需要快速完成演示，有以下选择：

#### 方案1：使用 IBP 方法（快10-20倍）
```bash
python train.py --lambda-weight 0.1 --bound-method IBP --epochs 5
python train.py --lambda-weight 0.5 --bound-method IBP --epochs 5
python train.py --lambda-weight 1.0 --bound-method IBP --epochs 5
```
- 每个训练约需 5-10 分钟
- 总时间约 15-30 分钟

#### 方案2：减少 epochs 数量
```bash
python train.py --lambda-weight 0.1 --bound-method CROWN-IBP --epochs 1
python train.py --lambda-weight 0.5 --bound-method CROWN-IBP --epochs 1
python train.py --lambda-weight 1.0 --bound-method CROWN-IBP --epochs 1
```
- 每个训练约需 30-60 分钟
- 总时间约 1.5-3 小时

#### 方案3：等待当前训练完成（推荐用于实际研究）
让当前的 CROWN-IBP 5-epoch 训练在后台继续运行，完成后依次运行后续两次训练。
